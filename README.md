# Constru-o-do-Primeiro-Chatbot-Inteligente-com-Python
Projeto de estudo para criar um chatbot inteligente com Python, integrando banco de dados, IA e NLP. Reúne vídeos, PDFs e tutoriais organizados no NotebookLM, com resumos, exercícios e testes de prompts. O objetivo é aprender na prática e desenvolver autonomia para construir assistentes virtuais funcionais.
